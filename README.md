# 👶 Daughter Name Picker

A lightweight, fun web app that helps you choose a baby name through simple A/B comparisons. Each round shows two randomly selected names — you pick the one you prefer until only one name remains. 🎉

---

## ✨ Features
- Randomized comparison of over 300 names  
- Clean and responsive design  
- Smooth animations  
- Confetti celebration when the final name is chosen  
- Live counter showing how many names remain  

---

## 🧠 How It Works
1. The app randomly selects two names from the list.  
2. You click the one you prefer.  
3. The unchosen name is removed from the pool.  
4. The process repeats until only one name is left.  
5. A final celebration reveals your chosen name. 🎊

---

- script.js → App logic + animations

## 🚀 Getting Started

Simply open the `index.html` file in any modern web browser. No installation or dependencies required.
